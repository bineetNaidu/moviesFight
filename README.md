# Movies Fight

Get all information of any Movies and compare. The site use [bulma.io](https://bulma.io/) as the css framework

## Installation

Clone the Repo.

```bash
pip install foobar
```

## Usage

_make sure to use you own apikey at index.js @line-13 && @line-50_

-   this is at line 13

```javascript
 params: {
  apiKey: __Your_API_KEY__,
  s: searchQuery,
  plot: true,
  },
```

-   this is at line 50

```javascript
params: {
  apiKey: __Your_API_KEY__,
  i: imdbID,
  plot: true,
  },
```
